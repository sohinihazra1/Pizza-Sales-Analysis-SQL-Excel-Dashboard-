# Pizza-Sales-Analysis-Using-SQL-Excel-Dashboard-
Pizza Sales Analysis (SQL + Excel Dashboard)

📌Overview

This project focuses on exploring pizza sales data to understand basic business patterns and performance.
It looks at how orders vary across time, product categories, and sizes, along with identifying high and low performing items.
The analysis is structured around a few key metrics and simple trend observations rather than complex modeling, built using SQL for data analysis and Excel for organizing and presenting the results.
 The focus is on deriving simple, structured insights from the data rather than applying complex techniques.

🛠️ Tools Used
- Microsoft SQL Server
- Microsoft Excel (Pivot Tables, Charts, Slicers)

📊 Key Analysis
- Revenue and order-based KPIs
- Daily and hourly sales trends
- Sales distribution by category and size
- Top and bottom performing pizzas

📸 Query Results
All SQL query results have been included as screenshots in this repository for reference. These screenshots show the outputs used to derive the KPIs and insights presented in the project.

📊 Dashboard
The Excel dashboard provides a visual summary of the analysis with interactive filters and KPI cards.

📌 Summary
This project demonstrates a basic end-to-end workflow of data analysis using SQL and Excel, focusing on clarity, structure, and practical understanding of the data.
